CSS

This should be the body for CSS.
Style tag refers to css. When tagging 'style' we can use css functions in webpage.
However, when style tag is in between HTML lines, it is one of the attributes of HTML. When style is tagged alone, it needs {} which indicates the attributes that belong in that specific function of CSS. {} is called 'Selector', and function inside it is called 'Declaration'.

Declaration : Declaration consists of property and value. For example, 'color' would be a property and 'red' would be a value.

**In the 7th session, we transformed all 'a' links to black and visited links to gray. For the link that's been currently visited, we turned it red.
*1st method
- To do this, we put '.' infront of the class to indicate that it's a 'class'. And then we put 'class =saw' in the HTML line
- To turn current link red, we put another class named 'active' in HTML line and add '.active' as red in css line
- The order of classes in css line matters, as classes written in HTML line follows last order in css line first.
*2nd method
- 'id' selector : put id='active' in HTML and indicate #active in css line. The order of reflection is id>class>tag. If the class of all the selectors are the same, the css line reflects the last written order.
- id should be the 'one and only' value. Therefore, value 'active' shouldn't be used again.
- Because of this hierarchy, we can put exceptions as id and put broad considerations as tag
